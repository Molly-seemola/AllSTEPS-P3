##  Project Overview

AllStepz is a modern, responsive e-commerce website specializing in affordable branded footwear. The website offers a seamless shopping experience across all devices, featuring popular brands like Nike, Adidas, Puma, and Jordan.

##  Project Purpose

This project was developed as part of the Web Development course to demonstrate:
- Responsive web design principles
- Interactive user interfaces
- Form validation and user feedback
- SEO optimization techniques
- Accessibility standards
- Shopping cart functionality

---

##  Features

### Core Functionality
- **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- **Product Catalog** - Browse shoes with search and filter capabilities
- **Shopping Cart** - Add items, view cart, and complete purchases
- **Form Validation** - Real-time validation with error messages
- **Interactive UI** - Smooth animations and user feedback
- **SEO Optimized** - Meta tags, keywords, and semantic HTML

### Pages Included
1. **Home Page** (index.html) - Hero section with featured products
2. **Products Page** (products.html) - Product catalog with search
3. **Cart Page** (cart.html) - Shopping cart and checkout
4. **Contact Page** (contact.html) - Contact form with validation
5. **Enquiry Page** (enquiry.html) - Product enquiry form
6. **About Page** (about.html) - Company information and team

---

##  Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Responsive styling with Flexbox and Grid
- **JavaScript (ES6+)** - Interactive functionality
- **LocalStorage API** - Persistent shopping cart
- **Font Awesome** - Icons and visual elements
- **Google Maps Embed** - Store location

##  SEO Implementation

### Meta Tags
- Descriptive meta descriptions on all pages
- Relevant keywords for search engines
- Proper title tags (50-60 characters)
- Open Graph tags for social sharing

### Image Optimization
- Descriptive alt text for accessibility
- Semantic file naming
- Optimized image sizes

### Semantic HTML
- Proper heading hierarchy (h1-h3)
- Semantic elements (header, nav, main, footer, article)
- ARIA labels for accessibility

---

##  Form Validation

### Contact Form
- Name: Minimum 2 characters
- Email: Valid email format (example@domain.com)
- Message: Minimum 10 characters
- Real-time validation with visual feedback

### Enquiry Form
- Name validation
- Email validation
- Product selection required
- Message validation

### Cart/Payment Form
- Full name validation
- Card number: 16 digits
- Expiry date: MM/YY format
- CVV: 3 digits
- Shipping address required

---

##  Shopping Cart Features

- **Add to Cart:** Select size and add products
- **View Cart:** See all items with prices
- **Remove Items:** Delete unwanted items
- **Total Calculation:** Automatic price calculation
- **Persistent Storage:** Cart saved in localStorage
- **Checkout:** Secure payment form simulation

---

##  Known Issues & Future Improvements

- Payment processing is simulated (not real)
